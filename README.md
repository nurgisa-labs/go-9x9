# Игра Go

Новый проект о настольной игре **го (бадук)**.

## Подтверждённый MVP

- **Web-приложение**.
- Доска **9×9**.
- **Два игрока на одном экране**.
- **Chinese area rules**: два последовательных паса, подсчёт живых камней и территории, коми **7,5** для белых.
- Без ИИ, онлайна, аккаунтов, рейтингов и внешних сервисов в первой версии.

Подробности и ещё не принятые правила зафиксированы в [Game Brief.md](Game%20Brief.md).

## Текущее состояние (2026-07-17)

Web-MVP реализован: игра 9×9 для двух игроков за одним экраном, Chinese area rules, взятия, отклонение самоубийства и повтора полной позиции, два последовательных паса, совместная разметка мёртвых групп, area scoring с коми 7,5 для белых, доступный русскоязычный браузерный интерфейс.

### Запуск

```
npm run dev
```

### Проверка (2026-07-17)

- `npm test -- --run` — пройдено: 6 файлов, 113 тестов.
- `npm run typecheck` — пройдено.
- `npm run build` — пройдено (13 модулей).
- Браузерная проверка: стартовый экран, легальный ход и смена очереди, отклонение хода в занятый пункт, два паса и переход к разметке мёртвых групп, разметка мёртвых, подтверждение подсчёта и результат. Взятия покрыты автотестами.

Зависимости — только TypeScript, Vite и Vitest; runtime/UI-фреймворков нет. ИИ, аккаунты, аналитика и постоянное хранение не добавлялись.
